# 系统简介
  MySpace项目愿景是每个人都能有一块属于自己的空间，不需要把数据托管到各种平台，熟人之间通过点对点方式交流，不再有中间平台。
## 项目功能
### 资源管理
+ 文章
+ 视频
+ todo
+ 评论
+ friend
+ ...
### 来宾管理
+ 搜索引擎
+ 匿名用户
+ guest
+ friends
+ own
+ institution
+ ...
